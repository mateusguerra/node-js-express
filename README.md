# node-js-express

A simple Node.js app using [Express 4](http://expressjs.com/).

## Running Locally

Make sure you have [Node.js](http://nodejs.org/) installed.

```sh
git clone git@github.com:mateusguerra/node-js-express.git
cd node-js-express
npm install
npm start
```

## Running Tests


```sh
npm test
```
